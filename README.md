# 🧩 Mini LeetCode — AI-Powered Coding Platform

A full-stack LeetCode-style web application where users can solve 
18 coding problems and get instant AI-powered code analysis using 
Google Gemini 2.5 Flash. Built with Java Spring Boot 3.2 backend, 
React 18 frontend and MySQL database.

## 🌟 What makes it special?
Unlike regular coding platforms, Mini LeetCode has a built-in AI 
Mentor that automatically analyzes your code after every submission 
and gives detailed feedback on correctness, time complexity, space 
complexity, edge cases and optimization tips.
